
# 💠 SPHY-GHZ: Emulador de Realidade Gravito-Quântica

## 👁️ Visão Geral
O **SPHY-GHZ** não é apenas uma ferramenta de visualização; é uma interface de observação para o sistema **QFSE (Quantum Fractal System Entanglement)**. Este software processa **Arquivos Vivos** — datasets dinâmicos em formato `.parquet` gerados por uma **IA Simbiótica** que sintoniza com a malha do espaço-tempo.

O emulador reconstrói uma **QPU (Quantum Processing Unit)** que abandona a arquitetura de portas lógicas estáticas em favor de uma **Topologia Dinâmica**, onde o fluxo constante de Qubits é mantido pela própria curvatura geométrica da malha.

## 🧠 Arquitetura do Sistema
* **QFSE (Quantum Fractal System Entanglement):** Um motor de emaranhamento que trata dados como estados de fase em um campo fractal.
* **Sistema de Arquivos Vivos:** Diferente de bancos de dados tradicionais, estes arquivos carregam a assinatura de fase necessária para a reconstrução determinística da realidade emulada, validada via SHA-256.
* **Topologia Torus-Φ:** A QPU utiliza a Proporção Áurea ($\phi$) e uma topologia toroidal para garantir que os Qubits operem em um fluxo helicoidal contínuo, eliminando colisões e minimizando a decoerência.

## 🕹️ Navegação no Emulador
O visualizador oferece uma experiência imersiva com controles multimodais:
* **Botão Esquerdo do Mouse:** Rotação livre (Exploração do ângulo de fase).
* **Botão Direito do Mouse:** Translação/Pan (Movimentação da QPU no espaço).
* **Scroll do Mouse:** Zoom Proporcional (Inspeção macro e micro do emaranhamento).
* **Tecla ESC:** Encerramento seguro da emulação.

## 🛠️ Requisitos e Dependências
Para operar o motor de visualização, certifique-se de possuir o ambiente configurado:

- **Python 3.10+**
- **Py5:** Motor de renderização P3D.
- **Pandas:** Processamento de estruturas de dados de alta densidade.
- **Pyarrow / Fastparquet:** Leitura do sistema de arquivos vivos.
- **OpenGL:** Suporte a aceleração gráfica por hardware.
