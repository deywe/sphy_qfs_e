# 🌀 SPHY 3D Wormhole: Sovereign Nodes Visualizer

## 📖 Overview

The **SPHY 3D Wormhole Visualizer** is a high-fidelity simulation designed to visualize the non-local transfer of **Sovereign Nodes**. By utilizing 3D rendering and real-time coordinate jumping, this tool demonstrates the "tunneling" effect—where matter/information ceases to exist at Point A and instantaneously reconstructs at Point B.

This visualizer is part of the **Harpia Quantum** research suite, focusing on deterministic data integrity and spatial-temporal folding.

## 🛠️ Technical Features

* **Quantum Jitter (Phase Vibration)**: Implements Brownian-style micro-oscillations (`vibe_x/y/z`) to simulate the uncertainty principle in high-energy states.
* **Wormhole Phase Rings**: Interactive 3D rings composed of "data cubes" that scale dynamically to represent the opening and closing of a spatial singularity.
* **Coordinate Jumping**: Unlike standard movement, the particles transition using an alpha-fade logic to represent true **Non-Locality**—there is no physical path traveled between origin and destination.
* **3D Material Rendering**: Uses `point_light` and `specular` mapping to give the 10 Sovereign Nodes a metallic, plasma-like appearance.

## 🚀 Getting Started

### Prerequisites

You will need Python installed along with the following libraries:

```bash
pip install py5 pandas pyarrow fastparquet numpy

```

### Setup

1. **Generate the Data**: Ensure you have a valid `sphy_wormhole_3d.parquet` file in the root directory (generated by the SPHY Kernel script).
2. **Run the Visualizer**:
```bash
python sphy_wormhole_3d_viewer.py

```



## 🎮 Interactivity

* **Rotation**: Left-click and drag the mouse to orbit around the wormhole.
* **Zoom**: Use the mouse wheel to move closer or further from the singularity.
* **Full Screen**: The application launches in full-screen mode to capture the high-resolution detail of the phase rings.

## 🧪 Scientific Philosophy

In this simulation, the 10 particles represent **Sovereign Nodes**—immutable pieces of information that maintain their identity (via SHA-256 validation in the kernel) despite the total collapse and reconstruction of their spatial coordinates.

The absence of a "travel trail" is intentional; it serves to challenge the classical view of speed ($v = d/t$) in favor of **Instantaneous Phase Shift**.
